StockLiser is a modern web application that combines fun and finance:

Generate random memes

Save memes to your personal gallery

Search real‑time stock prices

Maintain a personal watchlist

Fully authenticated user experience

This repository contains the frontend, built with React, Redux, and Material‑UI, and communicates with a Node.js backend.
